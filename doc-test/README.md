# doc-test fixtures

This folder contains synthetic and public-domain local-search fixtures for
validating Slack Beaver. Files are intentionally mixed across Markdown, TXT,
CSV, JSON, and JSONL so the agent can be tested against prose, poetry, market
tables, news-style briefs, dialogue transcripts, task lists, and quote
collections.

All market data, news briefs, conversations, and task items are fictional test
data. Public-domain literary excerpts use short searchable passages and local
Chinese translations for fixture validation. Do not treat these files as
authoritative editions, factual reporting, investment guidance, or operational
records.
